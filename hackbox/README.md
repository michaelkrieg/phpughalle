Hackbox
=======

Zum Loslegen solltest Du Vagrant und Virtualbox bereits installiert haben.
Dann einfach

    vagrant up

und warten :-)

Der Zugriff erfolgt dann mittels

    vagrant ssh
    ## dann optional:
    sudo su -

Die IP Adressen lauten: 172.21.77.100 (PHP 5.4) und 172.21.77.101 (PHP 5.5).
